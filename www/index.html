EvoThings App

Use this app with the EvoThings Workbench desktop application to develop mobile apps for IoT in HTML5/JavaScript. The desktop application is available at evothings.com.

How to use

Launch the EvoThings Workbench desktop application. Press the Scan button in this app. The running Workbench application should appear as a selectable item. Tap the item to connect.

Click the Run button of a project in the Workbench to launch an app on this device.

Alternatively, you can connect using the connect address displayed in the main window of the Workbench. Enter this address in the connect field and press the Connect button. Note that you don't have to enter the port number part of the address (:4042) when the standard 4042 port is used. In this case you can just enter the IP-address (e.g. 192.168.0.103).
